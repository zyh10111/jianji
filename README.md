# 简记（NoteEase）—— 智能语音笔记 App

一个基于 React Native + Expo 开发的跨平台移动应用，通过语音转文字技术帮助用户快速创建和管理笔记。

---

## 📱 项目简介

简记（NoteEase）是一款智能语音笔记应用，主要功能包括：

- 🎤 **语音转文字笔记**：用户说话，AI自动转成文字并保存为笔记
- 🏷️ **智能分类**：根据笔记内容自动分类（学习、生活、工作、灵感等）
- 🔍 **标签与搜索**：支持手动添加标签、按内容或标签快速搜索笔记
- ☁️ **数据同步**：云端存储，多设备同步（需简单后端支持）
- 📤 **导出分享**：笔记可导出为文本或分享到其他平台

---

## 🛠️ 技术栈

- **框架**：React Native + Expo
- **语言**：JavaScript (ES6+)
- **导航**：React Navigation
- **存储**：AsyncStorage（本地存储）
- **语音**：Expo AV（录音功能）
- **UI组件**：React Native 原生组件

## 📋 功能特性

### 核心功能

1. **语音转文字笔记**
   - 支持语音录制
   - 语音转文字（当前为模拟实现，可接入真实API）
   - 自动填充到笔记内容

2. **智能分类**
   - 自动分析笔记内容
   - 根据关键词匹配分类
   - 支持手动修改分类

3. **标签与搜索**
   - 支持手动添加标签
   - 系统自动提取标签
   - 全文搜索（标题、内容、标签、分类）
   - 分类筛选
   - 搜索结果高亮

4. **数据同步**
   - 本地存储（AsyncStorage）
   - 云端同步（模拟实现，可接入真实后端）
   - 同步状态提示

5. **导出分享**
   - 单个笔记导出
   - 所有笔记批量导出
   - 支持系统分享功能
   - 导出格式：文本文件（.txt）

### 辅助功能

- 笔记的CRUD操作（创建、读取、更新、删除）
- 笔记列表展示
- 笔记详情查看
- 下拉刷新
- 长按删除
- 友好的错误提示

## 📦 安装说明

### 环境要求

- Node.js >= 14.0.0
- npm 或 yarn
- Expo CLI（可选，推荐使用）
- iOS模拟器（Mac）或Android模拟器/真机

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd dazy
   ```

2. **安装依赖**
   ```bash
   npm install
   # 或
   yarn install
   ```

3. **启动项目**
   ```bash
   npm start
   # 或
   yarn start
   # 或
   expo start
   ```

4. **运行应用**
   - **iOS模拟器**：按 `i` 键或扫描二维码（需要Expo Go App）
   - **Android模拟器/真机**：按 `a` 键或扫描二维码（需要Expo Go App）
   - **Web浏览器**：按 `w` 键

### 使用 Expo Go App（推荐）

1. 在手机上安装 **Expo Go** App
   - iOS: [App Store](https://apps.apple.com/app/expo-go/id982107779)
   - Android: [Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent)

2. 启动项目后，扫描终端显示的二维码

3. 应用将在手机上打开

### 构建应用

#### Android APK

**方法一：使用EAS Build（推荐，最简单）**

```bash
# 1. 安装 EAS CLI
npm install -g eas-cli

# 2. 登录 Expo 账户（如果没有账户，请访问 https://expo.dev/ 注册）
eas login

# 3. 配置项目
eas build:configure

# 4. 构建 APK（预览版，可直接安装）
eas build --platform android --profile preview
```

**方法二：使用构建脚本（更简单）**

- **Windows**：双击运行 `构建APK.bat`
- **macOS/Linux**：运行 `bash 构建APK.sh`

**详细说明**：查看 [APK构建指南](docs/APK构建指南.md)

#### iOS IPA

```bash
# 构建 IPA（需要 Apple 开发者账户）
eas build --platform ios
```

## 🚀 使用指南

### 创建笔记

1. 点击首页右下角的 **"+"** 按钮
2. 输入笔记标题（可选）
3. 选择分类（学习、生活、工作、灵感、其他）
4. 输入笔记内容或点击 **"🎤 语音输入"** 按钮进行语音输入
5. 添加标签（可选）
6. 点击 **"保存笔记"** 按钮

### 查看笔记

1. 在首页点击笔记卡片查看详情
2. 在详情页可以编辑、分享或删除笔记

### 搜索笔记

1. 点击首页右上角的 **"🔍 搜索"** 按钮
2. 输入搜索关键词
3. 可选择分类进行筛选
4. 点击搜索结果查看详情

### 导出笔记

1. 在笔记详情页点击右上角的 **"分享"** 按钮，可分享单个笔记
2. 在首页点击右上角的 **"📤 导出"** 按钮，可导出所有笔记

### 同步数据

1. 点击首页右上角的 **"☁️ 同步"** 按钮
2. 当前为模拟实现，实际应用中需要接入真实后端API

## ⚠️ 注意事项

### 语音识别功能

项目已实现真实的语音识别功能，支持以下方式：

- **百度语音识别API**（推荐，国内使用）
  - 免费额度：每天50,000次调用
  - 中文识别准确率高
  - 配置说明：查看 [语音识别配置说明](docs/语音识别配置说明.md)

- **模拟实现**（默认，用于演示）
  - 无需配置即可使用
  - 返回示例文本

详细配置方法请查看：[语音识别配置说明](docs/语音识别配置说明.md)

### 云端同步功能

当前云端同步功能为**模拟实现**，实际应用中需要接入真实的后端API，如：

- Firebase
- Supabase
- 自建后端API
- 其他云服务

### 权限要求

应用需要以下权限：

- **麦克风权限**：用于语音录制
- **文件系统权限**：用于导出功能
- **网络权限**：用于同步功能（如果启用）
