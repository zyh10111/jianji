# 简记（NoteEase）—— 安装说明

## 📦 安装步骤

### 1. 环境要求

确保您的开发环境满足以下要求：

- **Node.js** >= 14.0.0
- **npm** 或 **yarn**
- **Expo CLI**（可选，推荐）
- **iOS模拟器**（Mac）或 **Android模拟器/真机**

### 2. 安装Node.js和npm

如果您的系统还没有安装Node.js，请访问 [Node.js官网](https://nodejs.org/) 下载并安装。

安装完成后，在终端中运行以下命令验证安装：

```bash
node --version
npm --version
```

### 3. 安装Expo CLI（可选但推荐）

```bash
npm install -g expo-cli
```

或者使用npx（不需要全局安装）：

```bash
npx expo --version
```

### 4. 克隆或下载项目

如果您使用Git：

```bash
git clone <repository-url>
cd dazy
```

或者直接下载项目压缩包并解压。

### 5. 安装项目依赖

在项目根目录下运行：

```bash
npm install
```

或者使用yarn：

```bash
yarn install
```

### 6. 启动项目

运行以下命令启动开发服务器：

```bash
npm start
```

或者：

```bash
yarn start
```

或者：

```bash
expo start
```

### 7. 运行应用

启动后，您会看到以下选项：

#### 方式一：使用Expo Go App（推荐）

1. **在手机上安装Expo Go App**
   - iOS: [App Store](https://apps.apple.com/app/expo-go/id982107779)
   - Android: [Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent)

2. **扫描二维码**
   - 启动项目后，终端会显示一个二维码
   - 使用Expo Go App扫描二维码
   - 应用将在手机上打开

#### 方式二：使用模拟器

**iOS模拟器（仅Mac）：**
```bash
# 在终端中按 'i' 键
# 或者运行
npm run ios
```

**Android模拟器：**
```bash
# 在终端中按 'a' 键
# 或者运行
npm run android
```

**Web浏览器：**
```bash
# 在终端中按 'w' 键
# 或者运行
npm run web
```

## 🔧 构建应用

### Android APK

1. **安装EAS CLI**
   ```bash
   npm install -g eas-cli
   ```

2. **登录Expo账户**
   ```bash
   eas login
   ```
   如果没有账户，请访问 [Expo官网](https://expo.dev/) 注册。

3. **配置项目**
   ```bash
   eas build:configure
   ```

4. **构建APK**
   ```bash
   eas build --platform android
   ```

5. **下载APK**
   - 构建完成后，按照提示下载APK文件
   - 将APK文件传输到Android设备
   - 在设备上安装APK（需要允许安装未知来源的应用）

### iOS IPA

1. **安装EAS CLI**（同上）

2. **登录Expo账户**（同上）

3. **配置项目**（同上）

4. **构建IPA**
   ```bash
   eas build --platform ios
   ```
   **注意**：构建iOS应用需要Apple开发者账户（$99/年）。

5. **下载IPA**
   - 构建完成后，按照提示下载IPA文件
   - 使用TestFlight或Xcode安装到设备

## ⚠️ 常见问题

### 1. 依赖安装失败

**问题**：`npm install` 失败或很慢。

**解决方案**：
- 检查网络连接
- 使用国内镜像源：
  ```bash
  npm config set registry https://registry.npmmirror.com
  ```
- 或者使用yarn：
  ```bash
  yarn install
  ```

### 2. 启动失败

**问题**：`npm start` 失败。

**解决方案**：
- 清除缓存：
  ```bash
  npm start -- --clear
  ```
- 删除node_modules并重新安装：
  ```bash
  rm -rf node_modules
  npm install
  ```

### 3. 无法连接模拟器

**问题**：模拟器无法启动应用。

**解决方案**：
- 确保模拟器已启动
- 检查模拟器版本是否兼容
- 尝试重启模拟器

### 4. 扫描二维码失败

**问题**：Expo Go App无法扫描二维码。

**解决方案**：
- 确保手机和电脑在同一网络
- 检查防火墙设置
- 尝试使用URL方式连接

### 5. 权限问题

**问题**：应用无法访问麦克风或文件系统。

**解决方案**：
- 在设备设置中授予应用相关权限
- iOS: 设置 > 隐私 > 麦克风 > 简记
- Android: 设置 > 应用 > 简记 > 权限
